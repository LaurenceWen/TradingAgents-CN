## 🎯 升级目标

### 核心目标

1. **可视化工作流** - 用户可通过拖拽界面定义分析流程
2. **智能体插件化** - 支持动态注册和配置智能体
3. **统一 LLM 层** - 消除 7 种 LLM 适配器的代码重复
4. **商业版分离** - 开源代码与商业代码清晰分离

### 非目标 (本期不做)

- MCP 协议支持 (等生态成熟)
- 自定义智能体上传 (下一版本)
- 工作流模板市场 (下一版本)

## 📁 目录结构设计

```
TradingAgentsCN/
│
├── 📂 tradingagents/           # Apache 2.0 开源 (保持)
│   ├── agents/                 # 基础智能体实现
│   ├── dataflows/              # 数据流处理
│   ├── graph/                  # 基础 LangGraph 结构
│   ├── tools/                  # 工具定义
│   ├── llm_adapters/           # 现有 LLM 适配器 (逐步弃用)
│   └── utils/                  # 工具函数
│
├── 📂 core/                    # 专有许可证 (新建) 🆕
│   ├── LICENSE                 # 专有许可证
│   ├── __init__.py
│   ├── workflow/               # 可视化工作流引擎
│   ├── llm/                    # 统一 LLM 客户端
│   ├── agents/                 # 智能体注册与高级实现
│   ├── prompts/                # 提示词管理增强
│   └── licensing/              # 授权管理
│
├── 📂 app/                     # 专有许可证 (已有)
│   └── routers/
│       └── workflow.py         # 工作流 API 🆕
│
├── 📂 frontend/                # 专有许可证 (已有)
│   └── src/views/
│       └── WorkflowDesigner/   # 可视化编辑器 🆕
│
└── LICENSE                     # 混合许可证
```
