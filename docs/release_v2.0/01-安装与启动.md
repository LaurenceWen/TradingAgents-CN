# 安装与启动

本文档适用于 TradingAgentsCN v2.0 Windows 安装版。

## 安装前准备

- 操作系统：Windows 10 或 Windows 11，64 位
- 磁盘空间：建议预留 6 GB 以上可用空间
- 网络：首次配置模型和数据源时需要联网

## 安装步骤

1. 运行安装程序 `TradingAgentsCNSetup-*.exe`
2. 按提示完成安装
3. 安装完成后，桌面或开始菜单会生成启动入口

## 首次启动

首次启动会自动完成以下动作：

- 解压运行文件
- 初始化本地运行环境
- 启动 MongoDB、Redis、后端和前端服务
- 导入默认系统配置
- 创建默认管理员账号

首次启动时间可能略长，请等待启动完成。

## 访问系统

启动完成后，在浏览器中打开：

- http://127.0.0.1
- 如果本机环境已配置为 localhost，也可访问 http://localhost

默认管理员账号：

- 用户名：admin
- 密码：admin123

## 常用启动脚本

安装目录根目录中通常会包含以下脚本：

- `start_all.ps1`：启动全部服务
- `stop_all.ps1`：停止全部服务
- `restart_all.ps1`：重启全部服务
- `start_services_clean.ps1`：以清理模式启动服务

如果桌面快捷方式无法使用，可以直接在安装目录中运行上述脚本。

## 首次登录后建议执行

1. 修改默认管理员密码
2. 配置至少一个可用的大模型 API Key
3. 根据需要配置股票数据源
4. 进行一次小范围分析，确认服务正常
5. 阅读 [05-服务管理与备份.md](./05-服务管理与备份.md) 了解后续维护方式
