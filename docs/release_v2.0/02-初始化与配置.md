# 初始化与配置

## 基础配置顺序

建议按以下顺序完成系统初始化：

1. 修改管理员密码
2. 配置大模型 API Key
3. 配置股票数据源参数
4. 验证分析与报告功能

## 管理员账号

系统首次安装完成后会创建默认管理员账号：

- 用户名：admin
- 密码：admin123

首次登录后请立即修改密码。

## 大模型配置

系统至少需要配置一个可用的大模型，常见选择包括：

- DeepSeek
- 阿里百炼 / 通义千问
- OpenAI
- Google Gemini

配置完成后，建议先执行一次简单分析任务，确认模型调用正常。

## 数据源配置

不同市场的数据源配置要求不同。常见情况如下：

- A 股：建议配置 Tushare Token
- 美股：根据所选数据源填写对应 API Key
- 港股：根据所选数据源填写对应参数

如果暂未配置完整数据源，也可以先使用基础功能验证系统是否可正常启动。

## 本地服务说明

安装版通常会启动以下本地服务：

- MongoDB
- Redis
- 后端 API 服务
- 前端 Web 服务

这些服务由安装版脚本统一管理，普通用户无需手动单独启动。

## 配置变更建议

- 修改关键配置后，建议执行一次重启
- 对生产使用环境，建议定期备份配置与数据目录
- 迁移到新机器前，先确认版本与配置兼容性
