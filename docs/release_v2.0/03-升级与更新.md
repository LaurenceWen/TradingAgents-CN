# 升级与更新

## 升级方式

TradingAgentsCN v2.0 当前支持两种升级方式：

1. 安装新版安装程序
2. 在系统内执行在线更新

## 推荐做法

对于普通用户，优先推荐使用新版安装程序完成升级。

原因：

- 操作更简单
- 对本地环境要求更低
- 适合从旧版本直接升级到新版本

## 在线更新说明

在线更新主要用于系统内的增量升级。

更新前建议：

1. 备份重要配置与数据
2. 确认当前系统未在执行长时间分析任务
3. 更新完成后按提示重启服务

可参考 [05-服务管理与备份.md](./05-服务管理与备份.md) 进行升级前备份。

## 升级后检查项

升级完成后，建议检查以下内容：

1. 能否正常登录系统
2. 历史配置是否保留
3. 大模型配置是否可正常调用
4. 分析任务是否可正常运行
5. 报告导出与通知功能是否正常

## 何时使用新版安装程序

以下情况建议直接使用新版安装程序：

- 当前版本跨度较大
- 本地环境曾做过较多手工修改
- 在线更新失败
- 需要重新整理本地运行环境

## 何时使用系统内更新

以下情况更适合使用系统内更新：

- 当前版本差异较小
- 只是常规修复或补丁更新
- 当前安装环境稳定且运行正常
