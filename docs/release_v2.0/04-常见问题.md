# 常见问题

## 启动后浏览器打不开页面

建议按顺序检查：

1. 稍等 10 到 30 秒，确认服务是否仍在启动
2. 访问 http://127.0.0.1
3. 如仍无响应，重启一次全部服务
4. 检查安装目录下 logs 目录中的日志文件

## 无法登录

请先确认：

- 是否使用默认管理员账号 `admin`
- 是否已经修改过密码
- 首次初始化是否已完成

如果忘记密码，建议联系维护人员或使用备份配置恢复。

## 模型调用失败

常见原因包括：

- API Key 未配置
- API Key 配置错误
- 账户额度不足
- 网络无法访问模型服务

建议先切换到另一模型提供商做一次验证。

## 数据获取失败

请检查：

- 数据源 Token 或 API Key 是否正确
- 当前网络是否可访问数据源接口
- 股票代码和市场是否填写正确

## 更新后功能异常

建议执行以下操作：

1. 重启全部服务
2. 再次登录并检查系统设置
3. 用一只测试股票做一次最小化分析
4. 如问题仍然存在，改用新版安装程序重新安装

## 日志在哪里

安装版运行日志通常位于安装目录下的 `logs` 目录。

排查问题时，优先查看：

- 后端日志
- 启动脚本日志
- 更新相关日志
