# v2.0.1 更新说明

发布日期：2026-03-11

本文档是面向最终用户的简版说明，只保留与使用体验直接相关的内容。

## 本次更新重点

1. 交易计划能力增强，规则生成、评估和优化建议更完整。
2. 数据同步稳定性提升，降低因数据状态异常导致分析不准的问题。
3. Windows 安装与自动更新流程完善，升级体验更顺畅。
4. 多项导出、持仓和显示问题得到修复。

## 用户可感知改进

### 交易计划

- 支持更完整的 AI 建议与规则优化流程
- 规则展示更直观，风险管理内容更易读
- 评估结果从单一总分升级为更细的维度展示

### 数据与同步

- 数据同步更稳
- 多数据源回退能力更好
- 降低“有数据但实际已不新鲜”的情况

### 导出与报告

- 报告导出能力增强
- Word / PDF / Markdown 导出可用性更好
- 一些字段映射和格式问题已修复

### 安装与更新

- Windows 安装版和更新链路更完整
- 升级过程中的配置保留和流程衔接更清晰

## 升级建议

如果你当前已经在使用 2.0.x：

- 可以优先使用系统内更新
- 升级后建议检查模型配置、数据源配置和一次测试分析

如果你当前版本较旧，或本地环境修改较多：

- 更建议直接安装新版安装程序

## 升级后建议自检

1. 登录系统是否正常
2. 模型是否可正常调用
3. 数据同步是否正常
4. 报告导出是否正常
5. 交易计划编辑与评估是否正常

## 说明

更完整的技术变更细节由内部发布说明维护；普通用户只需关注本页内容即可。
