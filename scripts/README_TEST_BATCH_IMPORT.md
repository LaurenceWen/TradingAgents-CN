# 批量导入API测试程序使用指南

## 📋 概述

`test_batch_import_apis.py` 是一个完整的测试程序，用于验证批量导入API的功能。它会：

1. 使用 Tushare 获取 000001（平安银行）和 000002（万科A）的数据
2. 通过批量导入 API 将数据导入到数据库中
3. 测试所有 5 种数据类型的导入功能

---

## 🚀 使用步骤

### 1. 确保 Tushare 已配置

检查 `.env` 文件中的 Tushare 配置：

```bash
TUSHARE_TOKEN=your_token_here
TUSHARE_ENABLED=true
```

### 2. 启动后端服务

在一个终端中启动后端服务：

```powershell
# 激活虚拟环境
.\env\Scripts\Activate.ps1

# 启动后端服务（本地开发环境使用8000端口）
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 3. 运行测试程序

在另一个终端中运行测试程序：

```powershell
# 激活虚拟环境
.\env\Scripts\Activate.ps1

# 运行测试程序
python scripts\test_batch_import_apis.py
```

## 📊 测试内容

### 测试的数据类型

| 序号 | 数据类型 | API 接口 | 数据来源 |

| 1 | 股票基本信息 | `/api/stocks/batch-import-basic-info` | Tushare `stock_basic` |

| 3 | 财务数据 | `/api/financial-data/batch-import` | Tushare 财务报表 |
| 4 | 新闻数据 | `/api/news-data/save` | Tushare 新闻接口 |
| 5 | 历史K线 | `/api/historical-data/batch-import` | Tushare `pro_bar` |

### 测试股票

- **000001** - 平安银行
- **000002** - 万科A

### 数据范围

- **新闻数据**: 最近 48 小时，每只股票最多 5 条
- **历史K线**: 最近 5 天的日线数据

## ⚠️ 注意事项

### 1. Tushare 权限要求

- 免费用户有调用频率限制（120次/分钟）
- 如果遇到频率限制，请等待1分钟后重试
- 建议升级到标准用户（800次/分钟）

### 2. 认证要求

- 批量导入功能为 **高级学员专属功能**
- 确保使用的账号是高级学员账号
- 默认使用 `admin/admin123` 登录

### 3. 数据来源标识

- 所有通过批量导入API导入的数据都会自动标记为 `local`（本地数据）
- 这样可以区分系统自动同步的数据和用户手动导入的数据

### 4. 去重规则

- **股票基本信息**: `(symbol, source)` - source 固定为 "local"
- **实时行情**: `symbol`
- **财务数据**: `(symbol, report_period, data_source)` - data_source 固定为 "local"
- **新闻数据**: `url`
- **历史K线**: `(symbol, trade_date, data_source, period)` - data_source 固定为 "local"

## 🔧 故障排除

### 问题 1: 无法连接到后端服务

**错误信息**: `Failed to establish a new connection: [WinError 10061]`

**解决方法**:
1. 确保后端服务已启动（本地开发环境使用 8000 端口）
2. 检查端口 8000 是否被占用
3. 检查防火墙设置

### 问题 2: Tushare 连接失败

**错误信息**: `Tushare 连接失败`

**解决方法**:
1. 检查 `.env` 中的 `TUSHARE_TOKEN` 是否正确
2. 检查 `TUSHARE_ENABLED=true` 是否设置
3. 检查网络连接

### 问题 3: 认证失败

**错误信息**: `登录失败`

**解决方法**:
1. 检查用户名和密码是否正确
2. 确保用户是高级学员
3. 检查数据库中的用户数据

### 问题 4: 权限不足

**错误信息**: `此功能为高级学员专属`

**解决方法**:
1. 使用高级学员账号登录
2. 或者升级当前账号为高级学员
